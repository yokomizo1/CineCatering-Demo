# CineCatering Demo

Demo publica e estatica do sistema de gestao operacional para catering em sets de filmagem.

Esta versao usa somente dados ficticios e roda sem backend, banco de dados, OpenAI, Supabase, PDFs, documentos privados ou integracoes reais.

## Demo

GitHub Pages:

https://yokomizo1.github.io/CineCatering-Demo/

## O que a demo mostra

- Dashboard executivo com indicadores operacionais ficticios.
- Financeiro com saldos, cartoes e fluxo projetado simulados.
- Ordens do Dia com cardapio, equipe e status.
- Estoque com alertas e compras previstas.
- Interacoes em modo demonstrativo, sem persistencia real.

## Seguranca

Este repositorio nao contem:

- Backend do produto.
- Arquivos `.env`.
- Chaves de API.
- URLs reais de banco.
- Dados financeiros reais.
- PDFs, contratos, notas, extratos ou documentos internos.

O produto completo permanece em repositorio privado.
